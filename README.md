# Nithish Kumar | Personal Portfolio 🚀

Welcome to the source code of my personal portfolio—built with a modern tech stack focused entirely on delivering a lightweight, zero-lag web experience featuring beautiful glassmorphism interfaces, interactive 3D elements, and highly optimized responsive architectures.

## 🌟 Features

- **Immersive 3D Experience:** Custom interactive Hero section featuring floating particle ecosystems utilizing React Three Fiber and Three.js.
- **Buttery Smooth Animations:** High-end motion design and scroll-driven parallax effects powered by GSAP & Framer Motion.
- **Premium Aesthetics:** A futuristic visual aesthetic utilizing subtle backdrop blurs, neon glow effects, and dynamic mesh gradients.
- **Lenis Smooth Scrolling:** Silky smooth scrolling experience across the entire page, providing a premium native-app feel.
- **Custom Interaction:** A highly optimized custom follower cursor adapting to specific hover states across the application.
- **Modern Performance:** Built entirely on top of React 18 & Vite, yielding lightning-fast Hot Module Replacement and highly minified production bundles.

## 🛠 Tech Stack

- **Core:** React 18, Vite
- **Styling:** Tailwind CSS V4
- **3D Graphics:** Three.js, React Three Fiber, React Three Drei
- **Animations:** GSAP, Framer Motion
- **Scroll Hijacking:** Lenis (Smooth Scrolling)

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You need [Node.js](https://nodejs.org/) and `npm` package manager installed on your machine.

### Installation

1. Clean install all dependencies:
   ```bash
   npm install
   ```

2. Start the local development server:
   ```bash
   npm run dev
   ```

3. Open your browser and visit: `http://localhost:5173`

## 📦 Build for Production

To create a highly optimized and minified production build:

```bash
npm run build
```

The output will be generated inside the `dist/` directory, ready to be deployed to your favorite hosting provider (Vercel, Netlify, Render, etc.).
